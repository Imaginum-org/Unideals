# 📌 How Boosts Work

- **Free Users**
  - No free monthly boosts.
  - Can purchase Boosts whenever needed.

- **Pro Users**
  - Receive **2 free Boost Credits every month**.
  - Can purchase additional Boosts after using all included credits.

- **Pro+ Users**
  - Receive **5 free Boost Credits every month**.
  - Can purchase additional Boosts after using all included credits.

# ✅ Monetization Philosophy

The Free plan should always provide a complete marketplace experience.

Premium plans should **help users sell faster and gain more visibility**, rather than restricting essential features.

Users should upgrade because they **want additional advantages**, not because basic functionality is locked behind a paywall.
